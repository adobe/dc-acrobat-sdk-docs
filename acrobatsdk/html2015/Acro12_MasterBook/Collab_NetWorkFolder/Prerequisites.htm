Prerequisites

The following are requirements for the comment repository server:

  • A network folder must be chosen to store the comments and it must be accessible and allow write access to all reviewers.

  • A local drive can be used to store the comments, but then comments can be created and viewed by only one user unless access to that drive is shared with the network.

The following are requirements for each client:

  • All users must have either Windows or Mac OS systems.
