CalCMYKProfile

The name of the ICC profile that is used for tagging or converting CMYK images, text, and/or graphics.

Note:Creative Suite applications give precedence to their own color management settings and may use this setting for compatibility with Distiller; see Using the color conversion settings for details.

Supported by

all applications

Type

string

UI name

Working Spaces: CMYK

Default value

(U.S. Web Coated SWOP v2)
