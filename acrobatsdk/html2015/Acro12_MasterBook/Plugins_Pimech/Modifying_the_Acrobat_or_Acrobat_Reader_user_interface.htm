Modifying the Acrobat or Acrobat Reader user interface

This section describes typical operations that a plug-in can perform to modify the Acrobat or Acrobat Reader user interface. To modify the user interface, you must invoke methods that belong to the Acrobat Viewer Layer. As a result, you cannot modify the Acrobat or Acrobat Reader user interface by using the PDF Library API. (See Acrobat Viewer layer.)
