SharePoint Services repositories

You can use a Microsoft Windows SharePoint Services document workspace as a comment repository for shared reviews. If your organization is already using SharePoint Services, you can leverage this resource by choosing it for the repository. Such a repository is easy to set up and configure, and is included with Windows Server 2003.

All reviewers must have read and write access to the workspace on the Windows server that is running SharePoint Services.

For more information on SharePoint Services, see http://www.microsoft.com/windowsserver2003/technologies/sharepoint/default.mspx.
