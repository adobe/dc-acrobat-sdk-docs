DDE Messages

This chapter lists all DDE messages supported by Acrobat.

These DDE messages handle the display of the Acrobat application:

  • AppExit 

  • AppHide 

  • AppShow 

  • CloseAllDocs 

  • HideToolbar 

  • MenuitemExecute 

  • ShowToolbar 

These DDE messages control the display of the document:

  • DocClose 

  • DocDeletePages 

  • DocInsertPages 

  • DocOpen 

  • DocReplacePages 

  • DocSave 

  • DocSaveAs 

  • DocSetViewMode 

  • FileOpen 

  • FileOpenEx 

These DDE messages handle printing of a document:

  • DocPrint 

  • FilePrint 

  • FilePrintEx 

  • FilePrintSilent 

  • FilePrintSilentEx 

  • FilePrintTo 

  • FilePrintToEx 

These DDE messages control the view of a document.:

  • DocGoTo 

  • DocGoToNameDest 

  • DocPageDown 

  • DocPageLeft 

  • DocPageRight 

  • DocPageUp 

  • DocScrollTo 

  • DocZoomTo 

This DDE message is used for searching:

  • DocFind 

Acrobat Reader supports the following subset of DDE messages:

  • AppExit 

  • CloseAllDocs 

  • DocClose 

  • DocGoTo 

  • DocGoToNameDest 

  • DocOpen 

  • FileOpen 

  • FileOpenEx 

  • FilePrint 

  • FilePrintEx 

  • FilePrintSilent 

  • FilePrintSilentEx 

  • FilePrintTo 

  • FilePrintToEx
