close all docs

Closes all documents.

Syntax

close all docs saving [constant]

Parameters

saving

Determines whether to save modified documents before closing. Possible values:

yes — Save the document.

no — Do not save the document.

ask — If the document has been modified, ask the user whether to save it.

The default value is ask.
