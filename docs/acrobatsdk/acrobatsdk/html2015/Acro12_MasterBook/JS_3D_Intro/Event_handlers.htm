Event handlers

There are several types of event handlers:

  • CameraEventHandler

  • KeyEventHandler

  • MouseEventHandler

  • MenuEventHandler

  • RenderEventHandler

  • ScrollWheelEventHandler

  • SelectionEventHandler

  • TimeEventHandler

  • ToolEventHandler

Each one responds to a different type of event during simulation. They use a callback mechanism to run a function when an event occurs. The event is passed as an argument to the event handlerâ€™s onEvent function so that it can be queried when the function runs. Event handlers are registered via the addEventHandler method, of the Runtime object.
