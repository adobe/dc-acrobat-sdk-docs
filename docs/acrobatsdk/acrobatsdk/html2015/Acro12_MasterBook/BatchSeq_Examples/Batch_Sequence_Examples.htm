Batch Sequence Examples

This chapter discusses some common tasks that can be done with batch processing, and offers sample solutions. Some of these solutions are fairly general, while others are specific to the system on which they were developedâ€”that is, they give specific paths, icons, and so on. You should study them carefully, and modify them to suit your needs.

Topics

Description

Count PDF files

Counts the number of PDF files to process, and sets the value of global.FileCnt.

Working with bookmarks

Sequences for dealing with bookmarks: gathering, listing, counting, copying and inserting.

Working with icons

Sequences for dealing with icon files: importing and creating navigation buttons.

Extract pages to a folder

For each file from the selected files, extracts each of its pages and saves the extracted pages to a particular folder.

Populate a form from a database and save it

Populates a form with values from a database, and saves the form to a folder.

Working with security

Security related tasks: removing security settings, getting signature information and signing.

Inserting comments and form fields

Tasks that insert an annotation or form field into a specific location on a page.

Spell check a document

Spell checks a document and gives suggested alternative spellings.

Summarize comments from selected files

Writes a report of all comments in the selected files.

Write comments to a tab-delimited file

Writes all comments in the selected files to a tab delimited file.
