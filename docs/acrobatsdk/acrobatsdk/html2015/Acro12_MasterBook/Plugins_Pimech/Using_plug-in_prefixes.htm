Using plug-in prefixes

It is important to correctly name all items located in your plug-in, such as HFTs, menus, toolbars, and so on, to ensure they function properly. Failure to do so may cause your plug-in to produce unpredictable results when your plug-in collides with a plug-in of another developer who used the same names.

Prevent spaces from being used in tokens you intend to use as names in a PDF file. This can happen, for example, if you allow a user to type a name into the PDF file and your plug-in does not check the input before writing it to the file.
