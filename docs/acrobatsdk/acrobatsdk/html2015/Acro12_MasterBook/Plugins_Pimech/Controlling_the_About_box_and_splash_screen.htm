Controlling the About box and splash screen

You can set values in the preferences file by invoking the AVAppSetPreference method to prevent the Acrobat or Acrobat Reader About box or splash screen from appearing before displaying the first document. These changes take effect the next time Acrobat or Acrobat Reader is started.

About Adobe Plug-ins is a standard menu command in the Help menu. This menu command contains a submenu. You are encouraged to have your plug-in add a menu command to the submenu to bring up its own About box.
