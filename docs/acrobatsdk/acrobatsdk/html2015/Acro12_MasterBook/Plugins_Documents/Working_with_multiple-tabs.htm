Working with multiple-tabs

In Acrobat DC, opening multiple PDFs in the same session opens the PDFs as tabs in the same application window. You can switch among the tabs from the top â€“ tab name shows the name of the file opened in the tab. You can use the Acrobat core API to navigate to a specific document tab or read name of a tab in Acrobat DC or Acrobat Reader DC. The method that navigate to a specific document tab is AVDocBringToFront and the method that reads name of a tab is AVDocGetDisplayTitle.
