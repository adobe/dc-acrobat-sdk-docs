Type

Required—The primary PDF datatype of the property. One of:

Fixed — Fixed-point number.

Int32 — A signed integer.

Atom — A PDF key (/XYZ).

String — A PDF string.

Color — An RGB color (array of three Fixed values).

BBox — A bounding box (array of four Fixed values).
