Forms API

The Forms plug-in for Acrobat allows plug-in developers to author Acrobat form fields.However, it cannot be used to edit LiveCycle Designer XML forms.

For Acrobat Reader, the Forms plug-in does not allow form authoring, but allows users to fill in data and print Acrobat forms. In general, the Acrobat Reader Forms plug-in does not allow users to save data to the local hard disk. However, if the PDF document has additional usage rights, then it may be able to save the document or perform other functions. For more information, see Rights-Enabled PDF Documents and Security.

Both Acrobat and Acrobat Reader allow web designers to send data from the form back to a web server.
