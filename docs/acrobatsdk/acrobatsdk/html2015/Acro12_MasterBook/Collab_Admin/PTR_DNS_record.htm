PTR DNS record

The first DNS entry needed to advertise the service is a PTR record that maps the Service Type to a particular service instance. In BIND the entry would appear as follows:

   <Service Type>.<Default DNS Domain>. IN PTR <Service Name>

The period (.) appearing after the Default DNS Domain must be included. Using the same example, the entry would appear as follows:

   _acrobatSRV._tcp.corp.example.com. IN PTR Example\032Service\032Name
