Tabbed interface for multiple PDFs viewing

Opening multiple PDFs in the same session now open the PDFs as tabs in the same application window. You can switch among the tabs from the top – tab name shows the name of the file opened in the tab. You can now have multiple AVDocs hosted in a single AVWindow. There is no longer a one-to-one mapping between AVWindows and AVDocs. So, every AVDoc does not have a unique AVWindow.

Note:The API AVDocGetAVWindow provides the AVWindow corresponding to an AVDoc, but it is not unique to the particular AVDoc

You can use the AVAppGetActiveDoc to get the current visible document or the document in the active and foremost tab. If Home or Tools tab is active, the API returns NULL.
