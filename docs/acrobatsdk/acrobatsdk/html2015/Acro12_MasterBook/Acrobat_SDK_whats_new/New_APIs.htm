New APIs

The following new APIs are added to the Acrobat and PDF Library API Reference.

Newly added API

Description

AVExtensionAcquireInfoEx

Fills the AVExtensionInfoRecEx structure with information about a client. It extends the AVExtensionInfoRec API which does not support new versioning scheme. It responsibility of caller to release the memory associated with the contents of the returned structure. You can use the AVExtensionReleaseInfoEx() method to release the memory.

AVExtensionReleaseInfoEx

Releases the memory associated with the contents of <code>info</code>.

AVDocBringToFront

Brings the specified document to front and makes its window and main view visible.

AVDocGetDisplayTitle

Gets the display-title of the specified document. Display title is the text that appears on application title bar of an active document.

AVIconCreateFromPDF

Creates a dynamically scalable icon from a PDF document.

AVToolBarSetIcon

Assigns icon to tool created using AVToolBarNew() method.
