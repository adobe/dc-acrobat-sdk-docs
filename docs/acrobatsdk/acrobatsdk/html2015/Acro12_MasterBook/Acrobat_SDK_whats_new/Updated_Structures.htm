Description

AVDocOpenParams

A new member variable AVDocOpenFlags openFlags is added. Set this variable to kAVDocOpenInNewWindow (1) to open the new document in a new window. By default, the value is set to kAVDocOpenWithNoFlags (0), which opens the AVDoc in new tab by default. The structure is used with API AVDocOpenFromFileWithParams. To use the structure, include the AVExpt.h header file.
