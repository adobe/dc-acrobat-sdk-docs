StRetrieve

StRetrieve restores the implicit parent stack from a saved state, whose name is specified by the StoreName key (as described in StStore on page 768). The syntax for a restoring the current state is as follows:

   [   /StoreName name
      /StRetrieve pdfmark

The previous state of the implicit parent stack is overwritten by the restored state. It is an error to try to retrieve a nonexistent state, that is, to use a name that was not associated with a stack state by a previous StStore.
