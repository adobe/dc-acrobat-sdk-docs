SetOCGIntent

The SetOCGIntent procedure sets the Intent key in ocgname to intent.

Syntax

ocgname intent SetOCGIntent

Parameters

ocgname 

Valid PostScript string variable representing an optional content group.

intent 

Value of ocgname’s Intent key, such as /Design, /View, /All, or /None, or an array of names, excluding /All and /None For a description of an optional content group’s Intent key, see the PDF Reference. The default value is /View.
