Acrobat Right Hand Pane (RHP) and App Center

The RHP accordion UI model of Acrobat is replaced with a more immersive tools experience. By default, a small set of tools is displayed in the RHP upon opening a PDF document. You can find your plug-ins and additional set of tools under the More Tools section in the RHP. Although the plug-ins now confer to new App framework, yet the plug-ins work as designed.

 

In Acrobat XI, tools of a plug-in appear in Right-hand pane (RHP).

 

In Acrobat, tools of a plug-in appear below the tools bar.

Acrobat DC tools by categories. Tools with similar functionalities and workflows are categorized together. Third-party plug-ins are listed in the Add-ons category. Custom tools are listed in the Customize category.
