SampleExtn

Location

Plugins/Samples/SampleExtn

Description

This sample demonstrates how a plug-in can work with Reader in the Sandbox mode ON. For details on how the Plug-in and Broker interact with each other and hence the Reader, please refer the section on ‘Sandbox Broker Extensibility’.

Usage

1. Configure the sample to enable it for "Adobe Reader" and build the plug-in.

2. Copy the SamplePI.api to <Reader_Installed_Path>\plug_ins\SamplePI.api.

3. Copy the SamplePIBroker.exe process to <Reader_Installed_Path>\plug_ins\pi_brokers\SamplePIBroker.exe.

4. Start Reader in Protected Mode and play with SamplePI.api from the Acrobat SDK->SamplePI menu.

Note:This is a Windows-only plugin.
