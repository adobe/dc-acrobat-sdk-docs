Including Acrobat DC SDK library files

To create a plug-in, you must include Acrobat DC SDK library files, such as header files, into your project. You can link to these library files from within your development environment. For more information, see the documentation that accompanies your development environment.

The Acrobat DC SDK library files are separated into the following categories:

  • Header files that are common to most plug-ins and generally referenced from PIMain.c.

  • Header files specific to core and extended APIs.

You can find these header files in the following directory:

  • Acrobat SDK\PluginSupport\Headers
