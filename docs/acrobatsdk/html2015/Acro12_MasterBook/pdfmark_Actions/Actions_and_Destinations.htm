Actions and Destinations

When a user opens a file, clicks on a link, or clicks on a bookmark, several types of information need to be specified to indicate what should happen. Different pdfmark types require one or more of the following:

  • Actions specify what type of action should be taken. They are indicated by the Action key in a pdfmark. See Actions. File specifiers indicate the target of an action when it is not the current file. See the table File specifier keys.

  • Destinations specify a particular location in a file, and a zoom factor. See Destinations.
