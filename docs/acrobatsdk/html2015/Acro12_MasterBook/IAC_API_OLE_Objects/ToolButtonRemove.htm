ToolButtonRemove

Removes the specified button from the toolbar.

Syntax

VARIANT_BOOL ToolButtonRemove(BSTR szButtonName);

Parameters

szButtonName

The name of the button to remove. See the Acrobat and PDF Library API Reference for a list of toolbar button names.

Returns

-1 if the button was removed, 0 otherwise.
