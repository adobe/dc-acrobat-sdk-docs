MenuItemIsEnabled

Determines whether the specified menu item is enabled.

Syntax

VARIANT_BOOL MenuItemIsEnabled(BSTR szMenuItemName);

Parameters

szMenuItemName

The language-independent name of the menu item whose enabled state is obtained. See the Acrobat and PDF Library API Reference for a list of menu item names.

Returns

-1 if the menu item is enabled, 0 if it is disabled or does not exist.
