goto

Displays the page that has the specified page number.

Syntax

goto [reference] page [integer]

Parameters

goto

The PDF Window object in which to change the page.

page

The page number of the page to be displayed. The first page in a document is page 1.

Related events

go backward 

go forward 

goto next 

goto previous 

AppleScript example

goto first PDF Window page 2

Apple event ID

kAEGotoPage ('gtpg')

Apple event parameters

keyAEPageNumber ('pg #')
