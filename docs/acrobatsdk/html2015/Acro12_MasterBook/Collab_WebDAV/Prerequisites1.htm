Prerequisites

The following are requirements for the comment repository server:

  • The latest version of Apache or Microsoft IIS.

  • A site to store the comments. This location must be accessible with read/write rights to all reviewers.

  • DAV access must be enabled for the root directory for the comments. For example, to add DAV access to an Apache server with mod_dav installed, add the following code to httpd.conf:

      <Location /Collab>

      Dav On

      </Location>

The following are requirements for each client:

  • Windows 7 or later, or Mac OS 10.6 or later.
