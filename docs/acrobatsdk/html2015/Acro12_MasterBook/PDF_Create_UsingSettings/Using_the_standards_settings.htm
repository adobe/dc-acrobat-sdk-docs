Using the standards settings

The Standards settings provide control over PDF/A- and PDF/X-compliant output:

  • PDF/A is a proposed ISO standard for the long-term preservation (archival) of electronic documents.

  • PDF/X is a focused subset of PDF designed specifically for reliable prepress data interchange. It is an International Standards Organization (ISO) standard (www.iso.org).

See Standards settings for details on the settings that are relevant to standards compliance.
