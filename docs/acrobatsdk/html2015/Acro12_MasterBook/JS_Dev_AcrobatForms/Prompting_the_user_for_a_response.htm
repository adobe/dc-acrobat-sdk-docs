Prompting the user for a response

Use the response defined in the app object. This method displays a dialog box containing a question and an entry field for the user to reply to the question. (Optionally, the dialog box can have a title or a default value for the answer to the question.) The return value is a string containing the userâ€™s response. If the user clicks Cancel, the response is the null object.
