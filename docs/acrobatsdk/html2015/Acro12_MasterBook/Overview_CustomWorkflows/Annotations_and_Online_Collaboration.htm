Annotations and Online Collaboration

This chapter describes how to use the Acrobat DC SDK to assist with annotations and online collaboration workflows.

Acrobat 8.0 introduces the Shared Reviews feature, which you should use for most online collaboration processes. You cannot initiate a Shared Review using the Acrobat DC SDK. For information on Shared Reviews, see the Acrobat Help.

If you need to create a custom collaboration environment, Acrobat also provides a platform for the development of collaborative review systems to fit various needs. For information about setting up a custom online collaboration system, see Acrobat Online Collaboration: Setup and Administration.
