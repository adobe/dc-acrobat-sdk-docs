Inserting Text into PDF Documents

This chapter explains how to use the PDF Library API to insert text into a PDF document. As stated earlier in this guide, the PDF Library API is a subset of the Acrobat core API that enables your application to interact and manipulate PDF documents. The PDF Library API is not used to create plug-ins for Acrobat Reader or Acrobat. It is also not used to modify the user interface of Acrobat Reader or Acrobat, such as by adding a toolbar or menu item.

Before reading this chapter, you should be familiar with creating a PDF Library application. (See Creating a sample PDF Library application.)
