Animation

A type of SceneObject, used to store keyframe animation sequences of Node objects in the Scene. In addition to the methods and properties below, it also contains the same methods and properties as a SceneObject.

Properties

Property

Type

Access

Description

number

R/W

The current time measured in seconds.

R

The end time of the sequence, measured in seconds.

number

R

The number of frames per second used to author the sequence.

length

number

R

The length of the Animation, measured in seconds.

startTime

number

R

The start time of the sequence, measured in seconds.
