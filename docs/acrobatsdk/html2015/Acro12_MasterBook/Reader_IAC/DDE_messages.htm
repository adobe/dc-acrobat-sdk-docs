DDE messages

Acrobat Reader supports the following DDE messages:

  • AppExit 

  • CloseAllDocs 

  • DocClose 

  • DocGoTo 

  • DocGoToNameDest 

  • DocOpen 

  • FileOpen 

  • FileOpenEx 

  • FilePrint 

  • FilePrintEx 

  • FilePrintSilent 

  • FilePrintSilentEx 

  • FilePrintTo 

  • FilePrintToEx
