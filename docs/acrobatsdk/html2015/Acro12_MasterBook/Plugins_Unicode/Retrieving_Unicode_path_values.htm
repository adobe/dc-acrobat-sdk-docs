Retrieving Unicode path values

You can use the Acrobat core API to retrieve a Unicode path value. The Unicode file system is essentially the same as the classic Windows file system except that its ASPathName object supports a few additional calls (through the file system call table) and the implementation uses the wide-char (Unicode) version of the Windowâ€™s APIs to access the native file system.

You can create an ASPathName object by using one of the following methods:

  • ASFileSysCreatePathName 

  • ASFileSysPathFromDIPathEx 

When you invoke either one of these methods, you must create an ASFileSys object to use as an argument.
