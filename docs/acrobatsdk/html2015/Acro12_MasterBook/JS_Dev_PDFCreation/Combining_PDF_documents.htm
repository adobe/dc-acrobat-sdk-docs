Combining PDF documents

You can customize and automate the process of combining PDF documents.

If you would like to combine multiple PDF files into a single PDF document, you can do so through a series of calls to the Doc objectâ€™s insertPages method.

  • Creating a new document from two other documents

   // Create a new PDF document:

   var newDoc = app.newDoc();

   

   // Insert doc1.pdf:

   newDoc.insertPages({

      nPage: -1,

      cPath: "/c/temp/doc1.pdf",

   });

   

   // Insert doc2.pdf:

   newDoc.insertPages({

      nPage: newDoc.numPages-1,

      cPath: "/c/temp/doc2.pdf",

   });

   

   // Save the new document:

   newDoc.saveAs({

      cPath: "/c/temp/myNewDoc.pdf";

   });

   

   // Close the new document without notifying the user:

   newDoc.closeDoc(true);
