PreflightProfile methods

hasConversion

9.0

 

 

P

Checks whether the preflight profile contains a conversion to a specific PDF standard.

Parameters

cVersion 

The PDF standard, represented as a string.

Valid standard keys:

  • PDF/X-1a:2001

  • PDF/X-1a:2003

  • PDF/X-3:2002

  • PDF/X-3:2003

  • PDF/X-4:2008

  • PDF/X-4p:2008

  • PDF/X-5g:2008

  • PDF/X-5n:2008

  • PDF/X-5pg:2008

  • PDF/A-1a:2005

  • PDF/A-1b:2005

  • PDF/E-1:2008

Returns

true if the specified conversion is configured in the profile, false otherwise.
