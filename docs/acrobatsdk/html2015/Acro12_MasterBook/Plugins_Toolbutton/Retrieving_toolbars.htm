Retrieving toolbars

You can use the Acrobat core API to retrieve an existing toolbar that appears in Acrobat Reader or Acrobat. After you retrieve a toolbar, you can perform additional tasks such as attaching a button. (See Attaching a button to a toolbar.)

You retrieve a specific toolbar by invoking the AVAppGetToolBarByName method. This method requires a constant character pointer that specifies the internal name of a toolbar and returns an AVToolBar object that corresponds to the toolbar. If the name cannot be found, this method returns NULL.

The following table lists toolbar names that appear in Acrobat and Acrobat Reader (Acrobat Reader does not contain all the toolbars that Acrobat does). The Display Name column specifies the toolbar name that appears in Acrobat or Acrobat Reader. The Internal Name column specifies the value that you must pass to the AVAppGetToolBarByName method to retrieve the toolbar.

The following code example retrieves the Tasks toolbar.

  • Retrieving a toolbar by name

//Retrieve the Tasks toolbar

const char * toolbarName= "Tasks" ;

AVToolBar myToolBar = AVAppGetToolBarByName(toolbarName);

Note:You can invoke the AVAppGetToolBar method to return an AVToolbar object that is based on the Advanced Editing toolbar.
