Using Acrobat Tracker to manage document reviews

Acrobat Tracker can be used to monitor PDF document reviews by publishing information using RSS 2.0, which is an XML-based format used for describing lists of items. It can also be used to produce a visual client for a web service application.

The Tracker URL API can be used to manage subscriptions through the use of application integration. This is accomplished with a specialized acrobat: notation, which uses XML extensions to customize the user interface. This syntax can be used to create links that add, update, or remove subscriptions to document reviews, open a PDF document in the browser, or display the Tracker window, as shown in the following examples:

Action

URL

Show Tracker

acrobat:Inbox?show

Add a feed to Tracker

acrobat:Inbox?addFeed=http://www.example.org/rss/example_rss.xml

Refresh the same feed in Tracker

acrobat:Inbox?refresh=http://www.example.org/rss/example_rss.xml

Select the same feed in Tracker

acrobat:Inbox?select=http://www.example.org/rss/example_rss.xml

Convert the same feed to PDF. If it is not currently subscribed, it is converted without adding it to Tracker.

acrobat:Inbox?convert=http://www.example.org/rss/example_rss.xml

For more information, see Acrobat Tracker.
