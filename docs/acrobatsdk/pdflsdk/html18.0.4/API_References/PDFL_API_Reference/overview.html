PDF Library API Reference

The PDF Library API Reference is one of several resources available to help you develop PDF Library applications.

What's in this guide?

This guide provides detailed descriptions for the APIs that can be used to develop PDF Library applications.

Who should read this guide?

This guide is meant for C/C++ developers who would like to develop PDF Library applications.

For information about the PDF Library SDK and the many ways that developers can build PDFL Library applications, see the Overview.
