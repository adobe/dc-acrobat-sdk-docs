About This Help

Welcome to the Adobe PDF Library SDK Help. This Help is intended for developers who create software that interacts with PDF technology.

Use this Help to learn about the following aspects of the PDF Library SDK:

•Understand the PDF Library SDK and its development technologies

•Create applications that directly create and manipulate PDF documents.

•Learn about PDF technology and recent standardization efforts.
